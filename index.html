Quantum computers are confusing, so let's start somewhere more familiar...
Classical Computing

Before we delve into the realm of quantum computing, let's first make sure we understand classical computing.

The modern definition for a computer is a type of electrical machine that can take an input, store it, process it, and produce an output.

Classical computers use logic gates composed of transistors to manipulate bits to process information.

Some important definitions:

A bit is the smallest unit of data that a computer can store. A bit can represent a variety of different binaries, but it can only store one of the two possible values within a particular binary in a given moment. Typically, this value is either a 1 or a 0.

Transistors are used to physically represent one of two binary values. Consequently, they are a component of all modern classical computers.

Logic gates are made up of a combination of transistors. They perform logical operations on binary data by taking one or more bits, and outputting a value or multiple values based on the arrangement of transistors within the logic gate itself.

These definitions will prove useful when we enter the spooky realm of quantum mechanics.
